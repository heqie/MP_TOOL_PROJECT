## V1.0
### 初版TOOL，功能如下：
1. 选择文件：
    - 设置指定sheet为MP Project List（internal），读取成功会显示文件路径，否则报错
2. 序列检查：
    - 以项目号为基准自增序号依次检查，项目号不同则序号会被判断应该不同（部分项目号有所差异但是同个序号的已在代码里进行了处理）
3. 等级检查：
    - 允许'v', 'A', 'V'，与空字符
4. IC型号检查：
    - 允许'7272', '7202H', '7202M', '7202H 7202M','7202M 7202H'，不允许空字符
5. 玻璃型号检查：
    - 见compare_information.xlsx，如有新玻璃可在该表中添加，不允许空字符
6. 接口定义检查：
    - 只允许SPI与I2C两种，不允许空字符
7. flash检查：
    - 对比接口定义，即SPI->N，I2C->Y，不允许空字符
8. TOOL版本检查：
    - 见compare_information.xlsx，如有新TOOL版本可在该表中添加
    - 如果站点为 "整机"，则 TOOL 版本可以为空或 "-"，如果站点不为 "整机"，且更新说明即P列有字符需要比对compare_information.xlsx&不可为空，否则可为空
9. 发布日期检查：
    - 允许格式YYYY.MM.DD或YYYY.M.D，若更新说明即P列有字符则不能为空
